# 🍓 ichiflow

> No relation with Homebrew.

AI-native enterprise workflows

## Documentation

Start with [`docs/README.md`](docs/README.md) — the navigable index to the architecture docs, ADRs,
and research inputs.

ichiflow is an AI-native enterprise workflow development framework. Most enterprise software is
back-office and customer-facing interfaces plus rule flows that determine outcomes plus manual
review; ichiflow modularizes that whole shape into a schema-centric, declarative-first, API-first,
pluggable, deeply auditable system. It is purpose-built so AI coding agents (Claude Code first) are
productive at both build time and run time.
